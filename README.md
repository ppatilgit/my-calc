# my-calc

Calculator for following calculations

2 or more operands:
--------------
Add
Multiply
Subtract
Divide
Percentage

1 operand:
----------
x² (square)
1/x
√ (srquare root)
∓ (negate)

Helper buttons:
---------------
AC - clears both equation and display/result
C - clears equation and uses result as firstoperand
⌫ - clears current operand value

Built With
--------
html
css/scss
javascript

How to Run?
----------
Clone and run html with any Live server
